# Page Templates｜页面模板

## Purpose｜用途

本页定义最重要的几类 wiki 页面模板。
这些模板的目标不是追求英文技术文档风格，而是让页面同时满足：
- 中文业务语义容易命中
- 英文路径和锚点稳定可复用
- agent 容易拆解、链接和组合

通用规则：
- 新增页面文件名优先中文
- 页面标题用中英混合
- 页面正文默认中文为主
- `aliases` 和 `keywords` 要优先覆盖中文问法
- 先写概览，再写细节
- 优先链接到表页、规则页和场景页，而不是在场景页里重复解释一切
- 如果是稳定概念或稳定对象，不要只留在场景页或规则页里，应考虑抽到 `concepts/` 或 `entities/`

## Bottom Line｜结论

如果某个页面不能完全套用某一个模板，至少保留这三点：
- 标题中英混合
- 中文检索别名完整
- 明确链接到相关对象页、规则页和场景页
