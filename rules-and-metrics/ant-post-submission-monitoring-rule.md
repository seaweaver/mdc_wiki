# Ant Post Submission Monitoring Rule｜蚂蚁提报后监控规则

## Overview｜概览

蚂蚁选品试点并不止于“能否提报”，还包括产品入池后的持续监控。

提报后监控的重点是：
- 是否触发出池预警
- 预警处于哪个阶梯
- 是否满足极端行情豁免

## Stage Logic｜阶段逻辑

工作表1定义了三类流程：
- 准入前：培育产品预警
- 准入前：提报产品预警
- 准入后：出池产品预警

本页聚焦第三类，即入池产品的持续监控。

## Performance Targets｜业绩考核目标

### 求赚系列

- 短债：最大回撤 `<=0.5%`，连续回撤天数 `<=5天`
- 中短债：最大回撤 `<=1%`
- 中长债：最大回撤 `<=2%`
- 极低波固收+：`3M<=1%`，`6-12M<=1.5%`
- 低波固收+：`<=2%`
- 中波固收+：`<=3%`
- 高波固收+：`<=5%`

### 求稳系列

- 短债：最大回撤 `<0.3%`
- 中短债：最大回撤 `<1%`
- 中长债：最大回撤 `<1.5%`
- 极低波固收+：最大回撤 `<1.5%`
- 低波固收+：最大回撤 `<2%`
- 中波固收+：最大回撤 `<3%`
- 高波固收+：最大回撤 `<5%`

## Drawdown Exemption｜回撤豁免条件

不同赛道对应的回撤豁免条件如下：

- 短债：近 x 年 / 成立最大回撤 `<=0.5%`，近 1 年连续回撤天数 `<5`，最大日回撤 `<0.15%`
- 中短债：最大回撤 `<=1.0%`，最大日回撤 `<0.20%`
- 中长债：最大回撤 `<=1.5%`，最大日回撤 `<0.50%`
- 极低波固收+：`3M-5M<=1.0%`，`6M以上<=1.5%`，最大日回撤 `<0.50%`
- 低波固收+：最大回撤 `<=2.0%`
- 中波固收+：最大回撤 `<=3.0%`
- 高波固收+：最大回撤 `<=5.0%`

极端行情豁免条件：
- 区间内基准指数突破回撤阈值
- 中证纯债基金指数在同一区间回撤天数 `>=5天`
- 天相短债 / 中短债 / 中长债基金指数最大日回撤超过阈值

## Warning Ladder｜分级触发阈值

### 第一阶梯

- 极低波：YTD 回撤 `0.5%`
- 低波：YTD 回撤 `1.0%`
- 输出：一级预警

### 第二阶梯

- 极低波：`1.0%`
- 低波：`1.5%`
- 中波：`2.0%`
- 高波：`4.0%`
- 极端行情补充：若基准指数年度最大回撤突破阈值，则按最大回撤排名后 `50%`
- 输出：二级预警

### 第三阶梯

- 极低波：`1.5%`
- 低波：`2.0%`
- 中波：`3.0%`
- 高波：`5.0%`
- 极端行情补充：若基准指数年度最大回撤突破阈值，则按最大回撤排名后 `70%`
- 输出：三级预警

## Monitoring Output｜预期输出

提报后监控结果理想上应包括：
- 当前所在赛道
- 当前预警阶梯
- 触发原因
- 是否满足极端行情豁免
- 建议动作
