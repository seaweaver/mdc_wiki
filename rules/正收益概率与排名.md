## R12 持有期正收益概率（绝对收益）

- rule_id: R12
- local_id: qudaojiankong:R12
- content_hash: `2b2b59148e360f20`
- use_cases: 步骤8

### 业务定义

滚动持有N个月，基金绝对收益 > 0 的概率。

### 参数表

| 字段 | 回看期 | 持有期 | 结束窗口类型 |
|:---|:---|:---|:---|
| P_2Y_HOLD_1M_POS | 2Y | 1M | ADD_MONTHS(start, 1) |
| P_2Y_HOLD_3M_POS | 2Y | 3M | ADD_MONTHS(start, 3) |
| P_3Y_HOLD_12M_POS | 3Y | 12M | ADD_MONTHS(start, 12) |
| P_2Y_ROLL_3M_POS | 2Y | 3M | ADD_MONTHS(start, 3) |
| P_2Y_ROLL_180D_POS | 2Y | 180D | start + 180 |
| P_2Y_ANY_HOLD_12M_POS | 2Y | 12M | ADD_MONTHS(start, 12) |
| P_2Y_HOLD_180D_POS | 2Y | 180D | start + 180 |
| P_3Y_HOLD_1Y_POS | 3Y | 12M | ADD_MONTHS(start, 12) |

### SQL 逻辑模板

```sql
-- 收益率计算
CASE WHEN NAV_START IS NOT NULL AND NAV_END IS NOT NULL AND NAV_START <> 0
     THEN NAV_END / NAV_START - 1 END AS RET
-- 有效性过滤: end_target_dt - end_dt > 15 天则剔除
WHERE (ADD_MONTHS(START_DT, HOLD_MONTHS) - END_DT) <= 15
-- 概率
ROUND(100 * SUM(CASE WHEN RET > 0 THEN 1 ELSE 0 END) / NULLIF(SUM(1), 0), 2)
```

## R13 同类排名（正收益概率）

- rule_id: R13
- local_id: qudaojiankong:R13
- content_hash: `56dc3d89f09f8f2d`
- use_cases: 步骤9

### 业务定义

按天相L4分类，同类内按正收益概率从高到低 DENSE_RANK 排名。

### 排名参数表

| 分类 | 回看期 | 持有期 | 排名字段(RANK) | 排名字段(STR) |
|:---|:---|:---|:---|:---|
| 短债 | 2Y | 1M | RANK_SHORT_P2Y_H1M_POS | RANK_SHORT_P2Y_H1M_STR |
| 中短债 | 2Y | 3M | RANK_MIDSHORT_P2Y_H3M_POS | RANK_MIDSHORT_P2Y_H3M_STR |
| 中长债 | 3Y | 12M | RANK_MIDLONG_P3Y_H12M_POS | RANK_MIDLONG_P3Y_H12M_STR |

排名逻辑与 R11 相同，区别在于排序依据从"超额收益概率"变为"正收益概率"。
