## R09 跑赢货币指数概率（超额收益+BP）

- rule_id: R09
- local_id: qudaojiankong:R09
- content_hash: `91fd8ecb9ddc2bff`
- related_dictionary: `跑赢阈值 (BP)`
- use_cases: 步骤6, 步骤7

### 业务定义

滚动持有N个月，基金收益 - 货币指数(h11025.CSI)收益 > 阈值(BP)，统计满足条件的滚动起点占比。

### 口径要点

- 货基指数: h11025.CSI 的 S_DQ_CLOSE
- 滚动窗口: 用 LAST_VALUE + RANGE BETWEEN 取窗口期末净值
- 有效性过滤: 目标结束日 - 实际最后交易日 > 15天则剔除该样本
- 阈值单位: BP = 万分之一（50BP = 50/10000）

### 阈值参数表（步骤6）

| 字段 | 回看期 | 持有期 | 阈值(BP) |
|:---|:---|:---|:---|
| P_2Y_HOLD_3M_OC50BP | 2Y | 3M | 50 |
| P_2Y_HOLD_3M_OC80BP | 2Y | 3M | 80 |
| P_3Y_HOLD_6M_OC120BP | 3Y | 6M | 120 |
| P_2Y_ROLL_12M_OC80BP | 2Y | 12M | 80 |
| P_2Y_HOLD_6M_OC20BP | 2Y | 6M | 20 |
| P_2Y_HOLD_12M_OC30BP | 2Y | 12M | 30 |
| P_2Y_HOLD_12M_OC80BP | 2Y | 12M | 80 |
| P_3Y_HOLD_12M_OC150BP | 3Y | 12M | 150 |
| P_3Y_HOLD_12M_OC200BP | 3Y | 12M | 200 |
| P_3Y_HOLD_12M_OC250BP | 3Y | 12M | 250 |

### SQL 逻辑模板

```sql
-- 滚动窗口期取值
LAST_VALUE(A.NAV_ADJ) OVER (
  PARTITION BY A.FUND_WINDCODE ORDER BY A.DT
  RANGE BETWEEN CURRENT ROW AND INTERVAL '3' MONTH FOLLOWING
) AS NAV_END_3M
-- 超额收益
(FUND_RET - CASH_RET) AS EXCESS
-- 概率 = 满足条件的起点数 / 有效起点总数 × 100%
ROUND(100 * SUM(CASE WHEN EXCESS > (50/10000) THEN 1 ELSE 0 END)
            / NULLIF(SUM(CASE WHEN ... THEN 1 ELSE 0 END), 0), 2)
```

## R10 货基指数收益率

- rule_id: R10
- local_id: qudaojiankong:R10
- content_hash: `ca22e2301cd02c0f`
- use_cases: 步骤6 (RET_2Y_HB, RET_2Y_HBNH, RET_3Y_HBNH)

### 业务定义

近2年/3年货币指数 (h11025.CSI) 的区间收益率和年化收益率。

### SQL 逻辑模板

```sql
-- 2年货币指数收益率
CASE WHEN CASH_START_2Y IS NOT NULL AND CASH_START_2Y <> 0
     THEN CASH_ASOF / CASH_START_2Y - 1 END AS CASH_RET_2Y
-- 年化
POWER(CASH_ASOF / CASH_START_2Y, 365 / (AS_OF_DT - CASH_START_DT)) - 1
```

## R11 同类排名（跑赢货币指数概率）

- rule_id: R11
- local_id: qudaojiankong:R11
- content_hash: `fd69c5e3515efd27`
- related_dictionary: `天相L4债券分类 (STYPE)`, `跑赢阈值 (BP)`
- use_cases: 步骤7, 步骤9

### 业务定义

按天相L4债券分类（短债/中短债/中长债），同类内按概率从高到低 DENSE_RANK 排名。

### 排名参数表

| 分类 | 回看期 | 持有期 | 阈值(BP) | 排名字段(RANK) | 排名字段(STR) |
|:---|:---|:---|:---|:---|:---|
| 短债 | 2Y | 3M | 50 | RANK_SHORT_P2Y_H3M_OC50BP | RANK_SHORT_P2Y_H3M_OC50STR |
| 中短债 | 2Y | 3M | 80 | RANK_MIDSHORT_P2Y_H3M_OC80BP | RANK_MIDSHORT_P2Y_H3M_OC80STR |
| 中长债 | 3Y | 6M | 120 | RANK_MIDLONG_P3Y_H6M_OC120BP | RANK_MIDLONG_P3Y_H6M_OC120STR |

### SQL 逻辑模板

```sql
DENSE_RANK() OVER (PARTITION BY STYPE ORDER BY WINRATE_PCT DESC NULLS LAST) AS PEER_RANK_RAW
COUNT(CASE WHEN SAMPLE_CNT > 0 THEN 1 END) OVER (PARTITION BY STYPE) AS PEER_TOTAL
-- 排名/总数
TO_CHAR(PEER_RANK_RAW) || '/' || TO_CHAR(PEER_TOTAL) AS 同类排名
```
