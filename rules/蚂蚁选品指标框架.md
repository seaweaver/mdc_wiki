# Ant Selection Metric Framework｜蚂蚁选品指标框架

## 概览｜Overview

蚂蚁选品规则目前可以归纳为六类指标层次：
- 基础门槛
- 基金经理资质
- 绝对收益与相对收益
- 持有体验
- 风险控制
- 仓位/结构与场景专属约束

这套页面用于提供赛道之间共享的指标语言和口径目录。

## 一级维度｜Core Dimensions

### 1. 基础门槛

核心指标：
- 成立时长
- 基金合计规模
- 同策略产品佐证

共性规则：
- 大部分赛道要求成立满 `2年/5个月` 或 `3年/5个月`
- 普通赛道基金规模一般要求 `>=2亿元`
- FOF 赛道基金规模要求 `>=0.5亿元`

### 2. 基金经理资质

核心指标：
- 投资年限
- 管理当前产品时长
- 同策略产品管理年限
- 公募经验（超额宝、增益宝等场景更关键）

共性规则：
- 管理当前产品通常要求 `>=5个月`
- 公募经验和投资年限要求随赛道不同而变化

### 3. 收益类指标

核心指标：
- 近1年收益
- 区间年化收益率
- 业绩基准达标
- 滚动超额收益胜率

典型表达：
- 跑赢货币基金 `+xxBP`
- 跑赢天相债基指数
- 跑赢中债 + 中证800 组合基准

### 4. 持有体验

核心指标：
- 持有期正胜率
- 月度/季度/年度正收益概率
- 封闭期正胜率

这些指标体现“用户持有体验”，是蚂蚁选品语境中的高频核心约束。

### 5. 风险控制

核心指标：
- 区间最大回撤
- 最大连续回撤天数
- 最大日回撤
- YTD 回撤

这是提报前准入与提报后预警之间最重要的桥梁。

### 6. 仓位/结构

核心指标：
- 平均含权仓位
- 单季报含权仓位
- 久期范围
- 持有期类型
- 封闭期类型
- 单一风险资产仓位（FOF）

## 指标说明｜Metric Notes

### 成立时长

业务含义：
- 从产品成立日到当前监控日的存续时长

共性口径：
- 以自然日计算
- 不满足 2/3 年但满 5 个月时，可通过同策略产品佐证进入人工复核路径

参见：
- [[concepts/成立时长]]

### 基金合计规模

业务含义：
- 基金最新披露的合计资产规模，含 A/C 等全部份额

注意事项：
- 当前规则表写明“取内部口径”
- 这意味着后续仍需单独补一页“规模取数口径”

### 区间年化收益率

业务含义：
- 区间收益经过年化转换后的收益率

当前文档已给出：
- `(区间收益)^(365/区间天数) - 1`

待补充：
- 区间收益的具体取值方法
- 净值缺失、成立以来替代、交易日映射规则

参见：
- [[concepts/区间年化收益率]]
- [[concepts/成立以来替代规则]]

### 业绩基准达标

业务含义：
- 在给定观察窗口内达到或跑赢业务指定基准

常见类型：
- 直接不低于天相债基指数
- 不低于固收+组合基准
- 跑赢货基加若干 BP

参见：
- [[concepts/业绩基准达标]]

### 投资年限

业务含义：
- 基金经理累计从事投资管理相关工作的年限

注意事项：
- 这项指标属于[[concepts/基金经理资质要求]]中的一个维度
- 它不能替代同一家族中的公募经验或当前产品管理时长要求

参见：
- [[concepts/基金经理资质要求]]

### 近1年收益

业务含义：
- 以当前判断日向前回看 1 年得到的累计收益

适用场景：
- 更适合稳健增值类赛道的近期收益要求

参见：
- [[concepts/近1年收益]]

### 月度/季度/年度正收益概率

业务含义：
- 按月、季、年等自然周期切分后，统计每个周期是否为正收益

适用场景：
- 更适合描述理财替代和收益稳定性要求

参见：
- [[concepts/月度季度年度正收益概率]]

### 滚动1年平均含权仓位

业务含义：
- 在滚动 1 年窗口内统计产品的平均权益暴露水平

适用场景：
- 更适合刻画固收+产品过去一年的稳定仓位风格

参见：
- [[concepts/仓位与暴露约束]]

### 滚动超额收益胜率

业务含义：
- 在给定统计区间内，所有滚动持有期中“跑赢基准 + 阈值”的概率

常见表达：
- 任意持有 3 个月跑赢货币基金 + 50BP 的概率
- 任意持有 1 年跑赢货基 + 200BP 的概率

### 持有期正胜率

业务含义：
- 任意时间买入并持有规定期限后，收益大于 0 的概率

这类指标在多个赛道反复出现，后续适合单独拆为概念页。

### 区间最大回撤

业务含义：
- 在统计区间内，净值从前期高点回落到低点的最大跌幅

当前表中给出的公式描述：
- `Max(1 - 当日净值 / 前期最高净值)`

### 最大日回撤

业务含义：
- 单日净值相较前一日的最大跌幅

当前表中给出的公式描述：
- `Max(1 - 当日净值 / 前一日净值)`

### 平均含权仓位

业务含义：
- 过去一段时间内的平均权益暴露

当前表中已明确：
- `(股票持仓市值 + 可转债持仓市值 × 0.5) / 基金总资产`

### 久期范围

业务含义：
- 定期报告披露的久期区间

当前规则表明确：
- 短债：`<=0.8年`
- 中短债：`(0.8, 1.5]`
- 中长债：`(1.5, 3]`

## 共性易错点｜Shared Gotchas

- 很多指标写了“近 X 年 / 成立以来”，成立不满年限时要切换到成立以来
- “取内部口径”类指标意味着规则页不能替代数据对象页
- “同类前 50%” 在部分地方是替代条件，但实现细则尚未补齐
- 部分赛道区分“全开 / 定开 / 持有期产品”，持有期结构是硬门槛，不只是备注
- 可转债在含权仓位中的折算系数当前为 `0.5`
- “成立以来替代”和“同策略产品佐证”都是历史不足时的补充路径，但不是同一条逻辑
