## R03 基金成立时长

- rule_id: R03
- local_id: qudaojiankong:R03
- content_hash: `5340b6b628d79e53`
- use_cases: 步骤1 — 产品信息

### 业务定义

从成立日到当前业务日期的月份数（TRUNC 取整）。

### SQL 逻辑模板

```sql
TRUNC(MONTHS_BETWEEN(
  TRUNC(TO_DATE(v_buss_date_8, 'yyyymmdd')),
  TO_DATE(D.F_INFO_SETUPDATE, 'YYYYMMDD')
)) AS SETUP_MONTHS
```

## R04 基金规模计算

- rule_id: R04
- local_id: qudaojiankong:R04
- content_hash: `2ab8cecbc4f0c12d`
- related_tables: `[[tables/非分级组合规模口径-A025_KB_ZTBH_SUM]]`, `[[tables/分级产品规模口径-V_VW_073_FJCP]]`
- use_cases: 步骤1 — 产品信息

### 业务定义

三种口径并存：
- **FUND_TOTAL_SCALE**: Wind 口径，取最新一期 F_PRT_NETASSET（200天窗口内）
- **FUND_TOTAL_SCALE_CJHX**: 我司实际口径，分分级/非分级
- **FUND_TOTAL_SCALE_AC**: AC 合计口径

### SQL 逻辑模板

```sql
-- Wind口径: 最近200天内最新净值日期的 F_PRT_NETASSET
SELECT F_INFO_WINDCODE, PRICE_DATE, F_PRT_NETASSET AS NETASSET_TOTAL,
       ROW_NUMBER() OVER (PARTITION BY F_INFO_WINDCODE
         ORDER BY PRICE_DATE DESC, ANN_DATE DESC NULLS LAST) AS RN
FROM STAGE.WIND2_CHINAMUTUALFUNDNAV
WHERE PRICE_DATE >= TO_CHAR(TO_DATE(v_buss_date_8, 'yyyymmdd') - 200, 'yyyymmdd')
-- 我司口径: 分级产品用 CJHX_DSCL.V_VW_073_FJCP，非分级用 CJHX_DW.A025_KB_ZTBH_SUM
```
