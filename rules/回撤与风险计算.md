## R14 最大回撤（MDD）

- rule_id: R14
- local_id: qudaojiankong:R14
- content_hash: `c8ba772173852812`
- use_cases: 步骤10

### 业务定义

在给定窗口内，从历史峰值到后续净值低点的最大跌幅。

### 口径要点

- 回撤 = 1 - NAV / PEAK_NAV（PEAK_NAV 为到当日为止的历史最高净值）
- MDD = MAX(回撤) over 窗口
- 窗口: 3M / 5M / 6M / 12M / 2Y / 3Y

### SQL 逻辑模板

```sql
-- 滚动峰值
MAX(NAV_ADJ) OVER (PARTITION BY FUND_WINDCODE, WIN
  ORDER BY DT ROWS BETWEEN UNBOUNDED PRECEDING AND CURRENT ROW) AS PEAK_NAV
-- 回撤
CASE WHEN PEAK_NAV IS NOT NULL AND PEAK_NAV <> 0
     THEN 1 - NAV_ADJ / PEAK_NAV END AS DRAWDOWN
-- MDD
MAX(DRAWDOWN) AS MAX_DRAWDOWN
```

## R15 最大日回撤

- rule_id: R15
- local_id: qudaojiankong:R15
- content_hash: `fbeca73e9ff6852f`
- use_cases: 步骤10

### 业务定义

单日净值跌幅的最大值。

### 口径要点

- daily_dd = 1 - NAV_t / NAV_{t-1}（仅当日收益<0时有意义）
- MAX_DAILY_DD = MAX(daily_dd) over 窗口

### SQL 逻辑模板

```sql
CASE WHEN PREV_NAV IS NOT NULL AND PREV_NAV <> 0
          AND NAV_ADJ / PREV_NAV - 1 < 0
     THEN 1 - NAV_ADJ / PREV_NAV
     ELSE 0 END AS DAILY_DRAWDOWN
```

## R16 最大连续回撤天数

- rule_id: R16
- local_id: qudaojiankong:R16
- content_hash: `a54f4ed7998e1e56`
- use_cases: 步骤10

### 业务定义

连续日收益为负的最大天数。

### 口径要点

使用 ROW_NUMBER 差值法（gap-and-island）分组统计连续负收益天数。

### SQL 逻辑模板

```sql
-- gap-and-island 方法
SELECT FUND_WINDCODE, (RN_ALL - RN_FLAG) AS GRP, COUNT(*) AS STREAK_LEN
FROM (SELECT ...,
      ROW_NUMBER() OVER (PARTITION BY FUND_WINDCODE ORDER BY DT) AS RN_ALL,
      ROW_NUMBER() OVER (PARTITION BY FUND_WINDCODE,
        CASE WHEN DAILY_RET < 0 THEN 1 ELSE 0 END ORDER BY DT) AS RN_FLAG
      WHERE DAILY_RET IS NOT NULL)
WHERE DAILY_RET < 0
GROUP BY FUND_WINDCODE, (RN_ALL - RN_FLAG)
```
