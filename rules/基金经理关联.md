## R05 同基金经理产品

- rule_id: R05
- local_id: qudaojiankong:R05
- content_hash: `3f916b6c16f58d55`
- use_cases: 步骤2

### 业务定义

取当前在任基金经理，找其管理的所有其他产品（不含自身），输出逗号分隔列表。

### 口径要点

- 基金经理标识: `COALESCE(F_INFO_FUNDMANAGER_ID, F_INFO_FUNDMANAGER)`
- 在职判断: 离职日期为 NULL / '00000000' 或 >= 业务日期
- 管理年限: `ROUND(MONTHS_BETWEEN(END_DT, START_DT) / 12, 2)`

### SQL 逻辑模板

```sql
-- 取当前在任的基金经理
WHERE NVL(TO_DATE(NULLIF(F_INFO_MANAGER_LEAVEDATE, '00000000'), 'YYYYMMDD'),
          DATE '2999-12-31') >= TRUNC(TO_DATE(v_buss_date_8, 'yyyymmdd'))
-- 聚合所有产品
SELECT BASE_WINDCODE,
       RTRIM(XMLAGG(XMLELEMENT(E, REL_WINDCODE || ',') ORDER BY REL_WINDCODE)
         .EXTRACT('//text()').GETCLOBVAL(), ',') AS PROD_LIST
```

## R06 投资年限与公募经验

- rule_id: R06
- local_id: qudaojiankong:R06
- content_hash: `901e20887fa6cf45`
- use_cases: 步骤3

### 业务定义

- **INVEST_YEARS**: 基金经理首次任职至今的年数（年，4位小数）
- **MANAGE_CURR_PRODUCT_MONTHS**: 管理当前产品的时长（月，取整）
- **PUBLIC_FUND_YEARS**: 公募经验年限（当前等同于 INVEST_YEARS）

### SQL 逻辑模板

```sql
ROUND(MONTHS_BETWEEN(TRUNC(TO_DATE(v_buss_date_8, 'yyyymmdd')),
       FIRST_MGR_START_DT) / 12, 4) AS INVEST_YEARS
TRUNC(MONTHS_BETWEEN(TRUNC(TO_DATE(v_buss_date_8, 'yyyymmdd')),
       CURRENT_START_DT)) AS MANAGE_CURR_PRODUCT_MONTHS
```
