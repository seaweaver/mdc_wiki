# Ant Chaoebao Rule｜蚂蚁超额宝规则

## 概览｜Overview

超额宝定位为理财替代，强调绝对收益、控制回撤和稳定持有体验。

当前子层级沿用 [[entities/超额宝层级体系]]：
- 月月
- 季季
- 年年-纯债
- 年年-极低波

## 基础门槛｜Base Thresholds

- 成立时长：`>=2年/5个月`
- 基金规模：`>=2亿元`
- 同策略产品：若提报产品成立不满 2 年，则同策略产品需由同一基金经理管理满 2 年，且提报产品已管理满 5 个月

基金经理要求：
- 月月：公募经验 `>2年`，管理当前产品 `>1年`
- 季季 / 年年：公募经验 `>3年`，管理当前产品 `>2年`

## 收益要求｜Return Requirements

“月月 / 季季 / 年年”这组业务层级的共性语义见 [[entities/超额宝层级体系]]，这里仅保留本轮赛道阈值。

### 绝对收益

- 月月：近 `12个月 / 成立以来` 月度正收益概率 `100%`
- 季季：近 `8个季度 / 成立以来` 季度正收益概率 `100%`
- 年年-纯债：近 `3年` 每年正收益
- 年年-极低波：近 `2年` 每年正收益

### 相对收益

- 月月：持有 `6个月` 战胜货基 `+20BP` 的概率 `=100%`
- 季季：持有 `1年` 战胜货基 `+30BP` 的概率 `=100%`
- 年年-纯债：持有 `1年` 战胜货基 `+80BP` 的概率 `>80%`
- 年年-极低波：持有 `1年` 战胜货基 `+80BP` 的概率 `>80%`

## 持有体验｜Holding Experience

- 月月：持有期正收益概率 `>=95%`
- 季季：持有期正收益概率 `=100%`
- 年年-纯债：持有期正收益概率 `=100%`
- 年年-极低波：滚持 `180天` 正收益概率 `100%`

## 风险控制｜Risk Control

- 月月最大回撤：`<=0.3%`
- 季季最大回撤：`<=1%`
- 年年-纯债最大回撤：`<=1.5%`

最大日回撤：
- 月月：近 `1年` 最大日回撤 `<=0.05%`
- 季季：近 `2年` 最大日回撤 `<=0.20%`
- 年年-纯债：近 `2年` 最大日回撤 `<=0.50%`

## 结构约束｜Structure Constraints

结构层级本身见 [[entities/超额宝层级体系]] 与 [[concepts/产品运作期限结构]]。

持有期：
- 月月：`1个月`
- 季季：`2-4个月`
- 年年：`6-12个月`

久期：
- 月月：`<=0.8年`
- 季季：`0.8年 < 久期 <= 1.5年`
- 年年-纯债：`1.5年 < 久期 <= 3年`
- 年年-极低波：`久期 <= 3年` 且含权仓位 `(0,5%]`
