## R07 月度/季度/年度胜率

- rule_id: R07
- local_id: qudaojiankong:R07
- content_hash: `06231c17c7d55f83`
- use_cases: 步骤4

### 业务定义

基于复权净值计算的滚动胜率指标。

### 口径要点

- 净值取法: 每月末最后一天的复权净值（优先 F_NAV_ADJUSTED）
- 月度收益率 = 本月末净值 / 上月末净值 - 1
- 月度胜率 = 近12个月中收益>0的月数 / 总月数 × 100%
- 季度胜率 = 近8个季度中收益>0的季度数 / 总季度数 × 100%
- 年度胜率 = 近2/3年中收益>0的年数 / 总年数 × 100%
- 盈利年份清单: LISTAGG 聚合所有收益>0的年份

### SQL 逻辑模板

```sql
-- 月度末净值
SELECT FUND_WINDCODE, TRUNC(TRADE_DT, 'MM') AS MONTH_KEY,
       MAX(TRADE_DT) AS MONTH_END_DT,
       MAX(NAV_ADJ) KEEP (DENSE_RANK LAST ORDER BY TRADE_DT) AS MONTH_END_NAV
-- 月度收益率
CASE WHEN LAG(MONTH_END_NAV) OVER (...) IS NOT NULL
     THEN MONTH_END_NAV / LAG(MONTH_END_NAV) OVER (...) - 1 END
-- 胜率汇总
ROUND(POS_COUNT / TOTAL_COUNT * 100, 2) AS WINRATE_PCT
```

## R08 收益率计算（Wind直取 vs 净值计算）

- rule_id: R08
- local_id: qudaojiankong:R08
- content_hash: `0695d89e792a8426`
- related_tables: `[[tables/中国共同基金业绩表现-ChinaMFPerformance]]`
- use_cases: 步骤5

### 业务定义

两种收益率来源互补使用：
- **Wind直取**: 从 ChinaMFPerformance 取 F_AVGRETURN_YEAR / F_AVGRETURN_TWOYEA / F_AVGRETURN_THREEYEAR
- **净值计算**: 取 AS_OF 日净值 / N年前最近净值 - 1
- **年化收益率**: POWER(nav_asof / nav_start, 365 / 区间天数) - 1

### SQL 逻辑模板

```sql
-- Wind直取 (取最新一期)
SELECT F_AVGRETURN_YEAR, F_AVGRETURN_TWOYEA, F_AVGRETURN_THREEYEAR, F_ANNUALYEILD
FROM STAGE.WIND2_CHINAMFPERFORMANCE WHERE RN = 1
-- 净值计算
CASE WHEN NAV_ASOF IS NOT NULL AND NAV_1Y IS NOT NULL AND NAV_1Y <> 0
     THEN NAV_ASOF / NAV_1Y - 1 END AS RET_1Y_CALC
-- 年化计算
CASE WHEN (AS_OF_DT - START_DT) > 0 AND (NAV_ASOF / NAV_START) > 0
     THEN POWER(NAV_ASOF / NAV_START, 365 / (AS_OF_DT - START_DT)) - 1 END
```
