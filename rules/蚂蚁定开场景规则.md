# Ant Dingkai Rule｜蚂蚁定开场景规则

## 概览｜Overview

定开场景是普通赛道规则的增强版。

准入前提：
- 满足基础赛道规则，或
- 满足定开产品特殊规则

定开场景的核心不是完全重写规则，而是基于封闭期特征，对部分风控约束给出豁免和阈值放松。

## 覆盖类型｜Covered Types

- 短债定开
- 中短债定开
- 中长债定开
- 低波固收+定开

## 基础门槛｜Base Thresholds

- 成立时长：
  - 短债 / 中短债：`>=2年`
  - 中长债 / 低波固收+：`>=3年`
- 基金规模：`>=2亿元`
- 基金经理：
  - 短债 / 中短债：投资年限 `>2年`，管理当前产品 `>5个月`
  - 中长债 / 低波固收+：投资年限 `>3年`，管理当前产品 `>5个月`

同策略产品：
- 表格当前文本中，中长债同策略条目仍写为“管理满2年”，建议后续复核与普通赛道是否一致

## 结构约束｜Structure Constraints

- 短债定开封闭期：`1个月`
- 中短债定开封闭期：`2-4个月`
- 中长债定开封闭期：`6个月及以上`
- 低波固收+定开封闭期：`6个月及以上`

额外仓位要求：
- 低波固收+定开平均含权仓位：`(0,8%]`

## 收益要求｜Return Requirements

- 短债定开：`>=天相短债基金指数`
- 中短债定开：`>=天相中短债基金指数`
- 中长债定开：`>=天相中长债基金指数`
- 低波固收+定开：`>=95% cba00121 + 5% 000906.SH`

## 持有体验｜Holding Experience

封闭期正胜率要求：
- 短债 / 中短债：`>85%`
- 中长债 / 低波固收+：`=100%`

滚动超额收益要求：
- 短债：任意持有 `3个月` 跑赢货币指数 `+50BP` 概率 `>70%` 或同类前 `50%`
- 中短债：任意持有 `3个月` 跑赢货币指数 `+80BP` 概率 `>70%` 或同类前 `50%`
- 中长债：任意持有 `6个月` 跑赢货币指数 `+120BP` 概率 `>70%` 或同类前 `50%`
- 低波固收+：任意持有 `1年` 正收益概率 `100%`

## 风险控制｜Risk Control

在定开场景下，区间最大回撤阈值已放松为：
- 短债定开：`<=0.63%`
- 中短债定开：`<=1.25%`
- 中长债定开：`<=1.875%`
- 低波固收+定开：`<=2.5%`

## 豁免逻辑｜Exemption Logic

满足封闭期正胜率条件时：
- 原始回撤阈值放松 `25%`
- 豁免滚动正胜率约束
- 豁免最大日回撤约束
