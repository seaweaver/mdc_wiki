# 数据对象目录说明｜Tables Catalog

## 用途｜Purpose

这个目录用于存放稳定、可被 agent 直接识别的数据对象资产。

典型对象包括：
- 表
- 视图
- 来源系统
- 平台
- 字段集合
- 映射表
- 枚举字典
- 标签对象

## 组织方式｜Organization

建议文件命名：
- `营销域数据对象总览.md`
- `人员归属规则表.md`
- `渠道网点信息表.md`

建议页面标题格式：
- `DWMS DIM RULES EMPLOYEE｜人员归属规则表`
- `CR TJJ QD QDXX｜渠道网点信息表`

## 编写规则｜Writing Rules

- 一个稳定对象原则上对应一个页面
- 除非是总览页，否则不要把多个无关表混在同一页
- 必须记录平台、粒度、关键字段和关联易错点
- 每个对象页都要链接到相关规则页和场景页

补充说明：
- 新增文件名优先中文
- 页面说明以中文为主
- 中文别名和关键词要覆盖业务同事的自然语言问法
