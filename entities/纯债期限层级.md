# Pure Bond Tenor Tiers｜纯债期限层级

## 概览｜Overview

纯债期限层级，是蚂蚁选品在纯债类产品里使用的一套稳定分类对象。

当前已知分为三层：

- 短债
- 中短债
- 中长债

它们不仅是名称不同，而是对应不同的久期区间、持有期结构、收益要求和风险阈值。

## 在当前规则中的作用｜Role in Current Rules

这套层级当前同时出现在：

1. 求赚纯债赛道  
   用于定义准入门槛、滚动超额收益门槛、持有体验和回撤约束。

2. 定开场景  
   短债定开、中短债定开、中长债定开沿用同一套期限语义，只是叠加了封闭期和豁免逻辑。

3. 提报后监控  
   求赚系列中的纯债部分，仍按短债、中短债、中长债分别设置考核目标和回撤豁免条件。

## 当前已知层级映射｜Known Mapping

### 1. 久期侧

- 短债：`<=0.8年`
- 中短债：`0.8年 < 久期 <= 1.5年`
- 中长债：`1.5年 < 久期 <= 3年`

### 2. 持有期侧

- 短债：全开 / `1个月`
- 中短债：全开 / `2-4个月`
- 中长债：全开 / `6个月及以上`

### 3. 基准侧

- 短债：天相短债基金指数
- 中短债：天相中短债基金指数
- 中长债：天相中长债基金指数

## 当前团队理解｜Current Team Understanding

当前可以把这套层级理解为“纯债产品风格的一级分类对象”。

也就是说：

- `久期范围` 解释的是层级边界怎么划
- `天相债基指数体系` 解释的是每一层对应什么外部基准
- 纯债规则页解释的是每一层在业务上要满足哪些门槛

## 易错点｜Gotchas

- 不能把短债 / 中短债 / 中长债只当成名称标签，它们背后绑定了完整规则集合
- 定开场景中的短债定开 / 中短债定开 / 中长债定开，本质上仍是同一套期限层级的封闭式变体
- 中长债在成立时长、经理年限、封闭期要求上通常比短债和中短债更严格
