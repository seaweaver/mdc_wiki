## 2. 术语定义 (Glossary)

* **确认金额 (confirm_amt)**: 交易确认成功的金额，不含手续费。
* **募集开始日期**: 产品发布的起始日期。
* **业务日期 (v_buss_date_8)**: 跑批基准日期，默认 T-1，格式 YYYYMMDD。
* **复权单位净值 (F_NAV_ADJUSTED)**: 考虑了分红再投资的净值，优先于单位净值使用。
* **单位净值 (F_NAV_UNIT)**: 未复权的原始净值，当复权净值缺失时回退使用。
* **产品代码 (PRODUCT_CODE)**: 格式为 6位基金代码 + '.OF'，如 '001199.OF'。
* **窄表 (T_DM_JKQD_FUND_METRICS)**: 中间工作表，13个 INSERT 步骤写入，随后 MERGE 到宽表。详见 `[[tables/蚂蚁渠道监控指标窄表-T_DM_JKQD_FUND_METRICS]]`
* **宽表 (DM_JKQD_FUND_METRICS)**: 最终目标表，渠道监控指标。详见 `[[tables/蚂蚁渠道监控指标宽表-DM_JKQD_FUND_METRICS]]`
* **基金池 (TMP_JKQD_FUND_METRICS)**: 基金入口临时表，控制计算范围。详见 `[[tables/基金池入口表-TMP_JKQD_FUND_METRICS]]`
* **跑赢概率**: 基金收益超越基准指数+BP阈值的滚动持有样本占比（%）。
* **正收益概率**: 基金绝对收益>0的滚动持有样本占比（%）。
* **同类排名**: 按天相L4债券分类（短债/中短债/中长债），同类内 DENSE_RANK 排名。
* **最大回撤 (MDD)**: Maximum Drawdown，区间内从峰值到谷底的最大跌幅。
* **组合久期 (DURATION_RANGE)**: 以债券市值为权重的持仓债券久期加权平均值。
* **含权仓位 (EQUITY_POS)**: 股票仓位 + 可转债仓位×0.5，反映基金的权益风险暴露。
* **有效样本过滤**: 目标结束日与实际最后交易日之差>15天，视为数据缺失，该样本不参与统计。
* **BP**: Basis Point，万分之一 (1BP = 0.0001 = 0.01%)。

## 4. 字段命名与精度规范

### 数值型精度约定

| 精度类型 | Oracle 类型 | 适用场景 |
|:---|:---|:---|
| 高精收益率 | `NUMBER(18,8)` | 收益率类字段 (RET_*, ANN_RET_*, MDD_*, MAX_DAILY_DD_*) |
| 百分比展示 | `NUMBER(9,2)` | 胜率/概率类字段 (WINRATE_*, P_*_POS, P_*_OC*BP) |
| 规模金额 | `NUMBER(20,6)` | 基金规模字段 (FUND_TOTAL_SCALE*) |
| 排名/比值 | `NUMBER(18,2)` | 同类排名数值 (RANK_*_POS, RANK_*_OC*BP) |
| 仓位精度 | `NUMBER(9,4)` | 含权仓位 (AVG_EQ_POS_4Q) |
| 整数计数 | `NUMBER(10)` | 时长/天数 (SETUP_MONTHS, MAX_CONSEC_DD_DAYS_*) |

### 字符串型格式约定

| 格式模板 | 示例 | 适用字段 |
|:---|:---|:---|
| 排名/总数 | `79/288` | RANK_*_STR |
| 产品代码列表(逗号分隔) | `001199.OF,001200.OF` | SAME_MGR_PRODUCTS |
| 产品:年限列表(逗号分隔) | `001199.OF:0.00,001200.OF:0.00` | SAME_MGR_PRODUCTS_MGR_YEARS |
| 正收益月/总月 | `9/12` | POS_MONTHS_12M_STR, POS_QTRS_8Q_STR |
| 盈利年份列表 | `2023,2024,2025` | PROFIT_YEARS_SINCE_INCEPTION |
| 季度仓位字符串 | `2025Q1:46.50%,2025Q2:46.55%` | EQ_POS_SINCE_2025_Q_STR |
| 期限+单位 | `3M`, `6M`, `12M`, `7D` | HOLD_PERIOD_TYPE, CLOSED_PERIOD_TYPE |

### 字段命名后缀约定

| 后缀 | 含义 |
|:---|:---|
| `_WIND` | Wind 直取值（非自主计算） |
| `_CJHX` | 我司内部口径 |
| `_AC` | AC合计口径 |
| `_PCT` | 百分比值 |
| `_STR` | 字符串格式 |
| `_POS` | 正收益相关概率 |
| `_OC` | 超额收益(outperform cash) |
| `_BP` | 阈值单位为 BP |
| `_DD` | 回撤(drawdown)相关 |
| `_MDD` | 最大回撤(Max Drawdown) |
| `_Y` / `_M` / `_D` / `_Q` | 时间窗口单位(年/月/天/季度) |
