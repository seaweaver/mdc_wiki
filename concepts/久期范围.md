# Duration Band｜久期范围

## 定义｜Definition

久期范围，是指产品组合在定期报告口径下应落入的期限区间，用于约束产品的利率风险暴露和风格分类。

## 为什么重要｜Why It Matters

在蚂蚁选品里，久期范围不是附属描述，而是区分短债、中短债、中长债、以及部分理财替代场景的重要结构条件。

## 当前规则中的典型区间｜Current Rule Thresholds

纯债赛道常见区间：

- 短债：`<=0.8年`
- 中短债：`0.8年 < 久期 <= 1.5年`
- 中长债：`1.5年 < 久期 <= 3年`

超额宝场景中也沿用了类似区间：

- 月月：`<=0.8年`
- 季季：`0.8年 < 久期 <= 1.5年`
- 年年-纯债：`1.5年 < 久期 <= 3年`

## 当前团队理解｜Current Team Understanding

久期范围本质上是一种风格约束。

它通常不会单独决定产品是否入池，但会和持有期类型、回撤约束、收益表现一起构成赛道定位。

## 易错点｜Gotchas

- 久期通常来自定期报告口径，不一定是日频可得
- “久期 <=3年” 与 “1.5年 < 久期 <=3年” 语义不同，不能混写
- 第二层实现时，需要确认是直接取披露值还是从底层债券仓位推算
