* **核心字段**:
  * `F_INFO_WINDCODE`: 基金 Wind 代码

  * `F_NAV_ADJUSTED`: 复权单位净值 (优先使用)

  * `ANN_DATE`: 公告日期 (用于去重取最新)
* **预设关联路径**:
  > **关联产品代码**:
  ```sql
  LEFT JOIN STAGE.WIND2_CHINAMUTUALFUNDNAV N
    ON N.F_INFO_WINDCODE = target.PRODUCT_CODE
   AND N.PRICE_DATE <= v_buss_date_8
  ```
  > **取复权净值(优先复权，回退单位)**:
  ```sql
  NVL(N.F_NAV_ADJUSTED, N.F_NAV_UNIT) AS NAV_ADJ
  ```
  > **取最新净值(按 ANN_DATE 去重)**:
  ```sql
  MAX(NVL(N.F_NAV_ADJUSTED, N.F_NAV_UNIT))
    KEEP (DENSE_RANK LAST ORDER BY N.ANN_DATE DESC NULLS LAST) AS NAV_ADJ
  ```

  ```sql
  -- 月度收益率

       THEN MONTH_END_NAV / LAG(MONTH_END_NAV) OVER (...) - 1 END
  -- 胜率汇总
  ```

* **口径要点**:
  - 期初净值: <= OPR_STARTDT 的最后一期净值（若缺失则取 >= OPR_STARTDT 的第一期）
  - 期末净值: <= OPR_ENDDT 的最后一期净值

  ```sql
        KEEP (DENSE_RANK LAST ORDER BY CASE WHEN PRICE_DATE <= OPR_STARTDT
          THEN PRICE_DATE ELSE '00000000' END),
      MIN(CASE WHEN PRICE_DATE >= OPR_STARTDT THEN NAV END)
        KEEP (DENSE_RANK FIRST ORDER BY CASE WHEN PRICE_DATE >= OPR_STARTDT
          THEN PRICE_DATE ELSE '99999999' END)) AS NAV_START
  -- 正收益概率
  ROUND(100 * SUM(CASE WHEN PERIOD_RETURN > 0 THEN 1 ELSE 0 END)
              / NULLIF(SUM(CASE WHEN PERIOD_RETURN IS NOT NULL THEN 1 ELSE 0 END), 0), 4)
  ```

- purpose: 来源章节为“2. 术语定义 (Glossary)”
- aliases: `2. 术语定义`, `2. 术语定义 (Glossary)`, `Glossary`
- representative_entries: 见原始事实摘录
- rules_or_constraints: 见原始事实摘录

#### 原始事实摘录

* **确认金额 (confirm_amt)**: 交易确认成功的金额，不含手续费。
* **募集开始日期**: 产品发布的起始日期。
* **业务日期 (v_buss_date_8)**: 跑批基准日期，默认 T-1，格式 YYYYMMDD。
* **复权单位净值 (F_NAV_ADJUSTED)**: 考虑了分红再投资的净值，优先于单位净值使用。
* **单位净值 (F_NAV_UNIT)**: 未复权的原始净值，当复权净值缺失时回退使用。
* **产品代码 (PRODUCT_CODE)**: 格式为 6位基金代码 + '.OF'，如 '001199.OF'。
* **窄表 (T_DM_JKQD_FUND_METRICS)**: 中间工作表，13个 INSERT 步骤写入，随后 MERGE 到宽表。
* **宽表 (DM_JKQD_FUND_METRICS)**: 最终目标表，渠道监控指标。
* **基金池 (TMP_JKQD_FUND_METRICS)**: 基金入口临时表，控制计算范围。
* **跑赢概率**: 基金收益超越基准指数+BP阈值的滚动持有样本占比（%）。
* **正收益概率**: 基金绝对收益>0的滚动持有样本占比（%）。
* **同类排名**: 按天相L4债券分类（短债/中短债/中长债），同类内 DENSE_RANK 排名。
* **最大回撤 (MDD)**: Maximum Drawdown，区间内从峰值到谷底的最大跌幅。
* **组合久期 (DURATION_RANGE)**: 以债券市值为权重的持仓债券久期加权平均值。
* **含权仓位 (EQUITY_POS)**: 股票仓位 + 可转债仓位×0.5，反映基金的权益风险暴露。
* **有效样本过滤**: 目标结束日与实际最后交易日之差>15天，视为数据缺失，该样本不参与统计。
* **BP**: Basis Point，万分之一 (1BP = 0.0001 = 0.01%)。
